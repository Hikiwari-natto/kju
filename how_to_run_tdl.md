# TDL (To-Do List) アプリの起動方法

このプロジェクトのアプリ（FastAPI）を起動して、ブラウザで表示する手順を説明します。

### 1. ターミナルを開く
PyCharmの下側にある「Terminal（ターミナル）」タブをクリックして開きます。
※DevContainerで開いている場合、既にコンテナ内の環境（`/workspaces/...`）に繋がっているはずです。

### 2. サーバーを起動するコマンドを入力
ターミナルに以下のコマンドを入力して、Enterキーを押してください。

```bash
python main.py
```

または、従来通り以下のコマンドでも起動できます。

```bash
uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

※ もしポート 8000 が既に使われているというエラーが出た場合は、このコマンドのように `--port 8001` と指定して起動してください。

#### コマンドの意味（初心者向け解説）
*   `python main.py`: Pythonを実行して `main.py` を動かします。これにより、自動的にサーバーが起動するように設定されています。
*   `uvicorn`: PythonでWebアプリを動かすためのサーバーソフトの名前です。
*   `main:app`: 「`main.py` というファイルの中にある `app` という名前のアプリを動かしてね」という意味です。
*   `--reload`: プログラムを書き換えたときに、自動的にアプリを再起動してくれる便利な設定です。
*   `--host 127.0.0.1`: あなたのパソコン（ローカルホスト）からのみアクセスを許可する設定です。

### 3. ブラウザで確認する
コマンドを実行すると、ターミナルに以下のようなメッセージが表示されます。
`INFO: Uvicorn running on http://127.0.0.1:8001 (Press CTRL+C to quit)`

ブラウザで以下のURLを開いてください。

[http://localhost:8001](http://localhost:8001)

### 4. 停止する方法
アプリを止めたいときは、ターミナルで **`Ctrl` キーを押しながら `C` キー** を押すと終了します。

これで、自分で作ったTODOリストアプリが実際に動いている様子を確認できます！
